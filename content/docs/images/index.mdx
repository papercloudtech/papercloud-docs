## Introduction

PaperCloud Cloud Images are pre-built virtual machine images for popular open source projects, ready to deploy on major cloud providers.

## Supported Platforms

- **AWS** — AMI (Amazon Machine Image)
- **Azure** — Managed Image
- **GCP** — Custom Image
